# Start TrotAnalyser

Procédure simple pour démarrer le projet.

## 1 — Cloner le projet

git clone https://github.com/Mat-gyver/trotanalyser

## 2 — Lancer l'application

chmod +x start.sh
./start.sh

Le script démarre automatiquement :

- le backend FastAPI
- l'application Expo

## Backend

API disponible sur :

http://localhost:8000

## Diagnostic du projet

Pour vérifier les dépendances Expo :

npm run check
